# 0004 — Static data contract

- Status: accepted

There is no shared static dataset. All user data is generated by the user
on their device and persisted in IndexedDB. The only "data files" shipped
with the app are:

- the seed dataset hard-coded in `src/lib/seed.ts`, which writes a small
  demo to IndexedDB on first run (idempotent — only seeds if `visits` is
  empty). The user can wipe it from Settings.
- room-tag URI scheme `rpm:<class>:<id>` is informal — any UTF-8 string is
  accepted by the scanner. Recommended for printed tags.

Peer-exchange payload contract (`AffinityVector`):

```ts
interface AffinityVector {
  rooms: Record<string /* room tag */, number /* 0..1 normalised */>;
  generatedAt: number /* unix ms */;
}
```

Only the normalised score is shared. Visit counts, dwell times, and
timestamps stay on the device.
